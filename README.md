# OpenModelica-Desktop-App
Desktop GUI application using PyQt6 to run OpenModelica simulation executables

# OpenModelica Desktop App (FOSSEE Screening Task)

## Description
This project is a Python-based GUI application built using PyQt6 that executes an OpenModelica compiled model.

## Features
- Select executable file
- Input start and stop time
- Run simulation using GUI

## Technologies
- Python
- PyQt6
- OpenModelica

## How to Run
1. Install dependencies:
   pip install PyQt6

2. Run application:
   python app.py

## Notes
- Model used: TwoConnectedTanks
- Due to model constraints, simulation may show warnings but executes successfully.
